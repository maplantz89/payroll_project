# payroll_project
Project simulating a payroll process for a sample group of employees.


Overview

This project simulates a payroll process for a sample group of employees. The goal was to calculate payroll accurately, validate data, and identify discrepancies in earnings and deductions.

Key Features
Calculated gross pay based on hours worked and pay rates
Applied federal and state tax estimates and benefits deductions
Validated payroll data to ensure accuracy from gross to net pay
Identified and corrected discrepancies in payroll calculations
Built a summary dashboard to analyze payroll expenses by department

Tools Used
Microsoft Excel
Pivot Tables
Data Validation Techniques

Key Skills Demonstrated
Payroll data processing
Data accuracy and verification
Financial data reconciliation
Reporting and analysis
